Frequently Asked Questions

Will it run on my Apple computer?

JobPro Directory runs on all (non-server) versions of Microsoft Windows including Windows 10. Windows emulators such as Parallels Desktop, VMWare Fusion, Wineskin Winery (free), VirtualBox (free), WinOnX and Boot Camp enable you to run JobPro Directory within Windows on your Apple computer.

Is a web version available?

No. JobPro Directory is designed for your Windows deskop to provide you with offline program functionality and integration with common Windows software like Microsoft Word, Excel and your preferred web brower while minimizing Google Search limitations for complex queries originating from a common IP address.

Does it run on servers?

For small installations, JobPro Directory is installed on each desktop by a Windows Administrator. For larger installations, an unattended, silient network installation package is available to administrators automating the deployment and updating of JobPro Directory. It does not run on servers.
